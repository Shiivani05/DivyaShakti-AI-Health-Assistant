🏥 DivyaShakti AI Health Assistant
An Intelligent Appointment Scheduling System for Small Healthcare Providers
📖 Abstract
The DivyaShakti AI Health Assistant is an AI-driven appointment scheduling system designed to automate and streamline the patient booking process in small clinics and healthcare centers. By leveraging workflow automation and conversational AI, the system eliminates manual intervention, reduces operational inefficiencies, and enhances patient experience.
📌 Problem Statement
Despite advancements in healthcare technology, many small clinics still rely on traditional appointment booking methods such as phone calls or in-person scheduling. These approaches suffer from:
Inefficient time management
Increased likelihood of human error
Lack of real-time availability tracking
Poor patient experience due to delays
💡 Proposed Solution
This project introduces an automated, AI-powered assistant that facilitates seamless appointment booking through a conversational interface on Telegram. The system is designed to:
Collect patient information interactively
Provide real-time slot availability
Automatically confirm appointments
Store and manage booking data efficiently
⚙️ System Architecture & Workflow
The system follows a modular and event-driven architecture:
User Interaction Layer
Patients initiate communication via Telegram
Processing Layer
AI Agent interprets user inputs
n8n orchestrates workflow automation
Scheduling Logic
Validates date and time inputs
Retrieves available appointment slots
Data Handling Layer
Stores appointment details in Google Sheets or a database
Response Layer
Confirms bookings and communicates with the user in real time
🧠 Technology Stack
n8n – Workflow automation and orchestration
Telegram Bot API – Conversational user interface
AI Agent – Natural language processing and interaction

✨ Key Features
Fully automated appointment scheduling
Real-time slot management
AI-based conversational interface
Platform-independent access (via Telegram)
Scalable architecture for multi-platform integration
